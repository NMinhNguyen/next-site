**Next.js** is the React framework for building JavaScript applications. It’s used and loved by <Link href="/showcase"><a>the world’s leading companies</a></Link>.

In this tutorial, you’ll learn Next.js basics by creating a very simple **blog app**. Here’s an example of the final result:

**[https://next-learn-starter.now.sh](https://next-learn-starter.now.sh/)** ([source](https://github.com/zeit/next-learn-starter/tree/master/basics-final))

By the time you complete all of this tutorial, you’ll understand:

- How to create an deploy a Next.js app to ZEIT Now
- How to create pages and do client-side navigation
- How to serve static assets, set metadata for each page, and use built-in CSS Modules support
- How to statically generate pages that require fetching external data
- How to create dynamic routes
- How to create API routes

### What You’ll Learn in This First Lesson

In this first lesson, you will:

- Create a Next.js app and deploy it to production using [ZEIT Now](https://zeit.co).
- Learn how to take advantage of Next.js’s hot reloading feature.
- Learn the basics of the **DPS** workflow (**D**evelop, **P**review, **S**hip) by making some edits and creating a preview deployment before shipping to production.

> This tutorial assumes basic knowledge of JavaScript and React. If you’ve never written a React component, you should go through [the official React tutorial](https://reactjs.org/tutorial/tutorial.html) first.
>
